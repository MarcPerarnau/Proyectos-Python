# 🐍 Programación Orientada a Objetos en Python

Esta carpeta contiene ejercicios prácticos de POO en Python, diseñados para aprender conceptos como clases, objetos, herencia y polimorfismo.

## 📂 Contenido

-  `person_class.py` → Clase Person con atributos y un método para saludar.
- `bank_account.py` → Clase BankAccount con métodos deposit y withdraw.
- `student_grades.py` → Clase Student que guarda notas en una lista y calcula el promedio.
- `car_class.py` → Clase Car con métodos para acelerar y frenar.
- `library_books.py` → Clases Book y Library para añadir y listar libros.
- `rectangle_area.py` → Clase Rectangle que calcula área y perímetro.
- `inheritance_shapes.py` → Clases Shape, Circle y Square usando herencia.
- `employee_salary.py` → Clase Employee que calcula salario según horas trabajadas.
- `shopping_cart.py` → Clase ShoppingCart con productos y total de compra.
- `animal_polymorphism.py` → Polimorfismo con clases Dog y Cat que heredan de Animal.

▶️ Ejecución

Ejemplo de ejecución:
```
python poo/person_class.py
```
📌 Notas

- Todos los ejercicios usan clases para modelar objetos del mundo real.

- Son ejemplos educativos, no incluyen tests automatizados.
